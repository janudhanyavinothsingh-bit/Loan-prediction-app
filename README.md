#  Loan Prediction App

This project is part of my learning journey in machine learning and deployment.  
It predicts whether a loan application will be approved based on applicant details like income, dependents, loan amount, and CIBIL score.  

I wanted to go beyond just training a model in Jupyter Notebook — so I built a Streamlit app to make the predictions interactive and easy to use.

---

##  Why This Matters
Loan approvals are a big deal in real life. Behind the scenes, banks use models and rules to decide who qualifies.  
By recreating a simplified version, I learned how data, features, and algorithms come together to make decisions that impact people’s lives.

---

##  What You’ll Find Here
- **Interactive App (`app.py`)** → A Streamlit app where you can enter applicant details and get predictions.  
- **Trained Model (`loan_prediction_model.pkl`)** → A Random Forest model saved with joblib.  
- **Notebook (`loan_approval_dataset_ml.ipynb`)** → My Jupyter Notebook with data exploration, training, and evaluation.  
- **Dependencies (`requirements.txt`)** → Everything you need to install and run the app.  

---

##  How to Run It
Clone the repo:
pip install -r requirements.txt
Run the app:
streamlit run app.py
